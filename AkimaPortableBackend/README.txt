# Akima Portable Backend (v1)

This is a standalone version of the Akima music backend.

## How to use:
1. Double-click "launcher.vbs".
2. The backend will start in the background (no window will appear).
3. To connect your phone app, visit your computer's IP address on port 3000 in your phone's browser.

## Security:
- The source code is obfuscated and bundled into the executable.
- No Node.js installation is required.
